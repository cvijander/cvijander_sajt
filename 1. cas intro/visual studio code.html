Visual studio editor

Visual Studio Code editor ili skraceno VS Code je besplatni editor iz grupe mikrosoft alata. Predstavlja besplatan a snazan alat koji nam pomaze i omogucava lakse programiranje i pravljenje stranica.

Instalacija i preuzimanje

VS Code editor nalazimo na stranici Visual studio code, gde je postupak instalacije jednostavan i intuitivan. Pocinjemo se instalacijom koristeci dugme Download, sacekamo da on preuzme. Zatim se instalira klikom na ikonicu bukvalno pratimo instrukcije oko instaliranja. Nakon instalacije dobijamo otvoren novi program sa pocetnom stranom.
