HTMl

Html je svuda oko nas, primenjuje se u web sajtovima, aplikacijama,u drugim softverima i u svim primerima gde koristimo web tehnologiju.Html predstavlja osnovu za sadrzaj koji cemo imati na sajtovima, aplikacijama. Kao sto su tekst, slike, video, audio, forme i razne interaktivna desavanja na internetu.


Html predstavlja spregu izmedju izmedju jezika koji je razumljiv ljudima sa jedne strane i racunarima sa druge strane, tako sto sadrzaj markapuje tj obelezava ono sto se nalazi izmedju tagova i cime objasnjava racunara kako da se sa tim obelezenim kodom ponasa.Html jednostavno deklarise sta koja linija koda treba da predstavlja i to posle browser pokazuje, ako i postoji koja greska u kodu, to ce browser i nekako preci preko iste i prikazati.Jednostavnost html daje mu otpornost na vreme, tako da sajt koji je radjen pre par godina jos uvek se moze prikazivati na vecini browsera.

Sveto trojstvo

Web je sacinjen od 3 programska jezika, Html, Css, JavaScript.


Html je skracenica od Hyper Text Markup Language - tj laicki prevod bi predstavljao jezik koji velikom brzinom iscitava obelezeni deo. Html definise strukturu sajta.


Css skraceno od cascading style sheets -predstavlja kaskadni oblik teksta koji definise stilove na sajtu.Boja, tipografija, velicina, dakle izgled celog sajta definisemo preko cssa,cak je moguce dodati i jednostavne animacija i interakcije na sajtu koristeci css.Css je vise kompujterizovani jezik tj sadrzi vise logike u odnosu na html, ali u isto vreme je sklonijim mogucim greskama.Kada postoji greska u css kodu, browser ce taj deo koda preskociti i nastaviti dalje.


JavaScript je programski jezik koji poseduje mogucnost da kreiramo znacajniju interaktivnost.Ona je najjaca od svih prethodnih jezika, dakle od htmla i css, dosta stvari JavaScript moze da uradi koje nazalost ne mogu da izvrse html i css.Ali je takodje najvise podlozna greskama, ako browser ne moze da procita ili zbog starosti ili zbog nekog drugog detalja, jednostavno browser ce taj deo odustati od ucitavanja, cime cini JavaScript opet najtezim za citanje i za razumevanje. Postojanje sva tri jezika HTML,CSS i JS daje sajtu veliku moc, kombinujuci njihove prednosti kao sto su rezilijentnost, robusnostnost i snagu i smanjivajuci mane, tako da je preporuka da se ima sto vise html koda, pa css pa js zbog nazalost vecih gresaka koje mogu se pojaviti kod koriscenja js.
